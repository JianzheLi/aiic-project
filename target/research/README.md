# Research 总入口

## 研究目录

- `xhs-tech-intern-interviews/`：第一轮小红书公开内容调研，验证真实用户痛点和面经表达。
- `interview-questioning-playbook/`：第二轮广泛公开来源调研，提炼面试官追问链、八股/专项图谱和竞品初步观察。
- `demand-validation-desk-research/`：第三轮需求验证 desk research，补项目样本、岗位 JD 和竞品体验。
- `synthesis.md`：三轮研究整合结论，直接服务产品设计、Product Memo 和实现取舍。

## 当前产品判断

首版产品应聚焦“中国本科生技术实习面试准备”，核心闭环是：

1. 用户输入项目经历。
2. 选择训练场景。
3. AI 面试官连续追问。
4. 用户逐轮回答。
5. AI 输出结构化挂点复盘和下一轮练习建议。

最值得首版实现的场景是：项目深挖压力面、后端八股项目化追问、RAG/Agent 项目真实性拷打。
