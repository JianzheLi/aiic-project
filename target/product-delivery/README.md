# Product Delivery

本目录保存 AI 模拟面试官挑战的提交材料草稿。

## 文件

- `product-memo.md`：Product Memo 草稿。
- `product-memo.tex` / `product-memo.pdf`：最终提交用 Product Memo，LaTeX 排版，控制在 1-2 页。
- `demo-slides.tex` / `demo-slides.pdf`：基于 PKU Beamer 模板的 Demo 展示 PPT。
- `demo-video-script.md`：3 分钟以内 Demo 视频脚本。
- `sample-resume/fake-agent-resume.tex` / `sample-resume/fake-agent-resume.pdf`：用于 demo 的脱敏 Agent/RAG 假简历。
- `architecture-diagram-prompt.txt`：用于生成 Agentic RAG 架构图的图像模型 prompt。
- `validation-checklist.md`：本地、Docker、公网验证清单。

## 当前产品一句话

面向中国本科生技术实习面试准备的 AI 模拟面试官：上传 PDF/DOCX/TXT 简历，获得连续追问和结构化挂点复盘。
