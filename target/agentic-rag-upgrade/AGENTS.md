# AGENTS.md

## 职责

本目录记录“Agentic RAG 面试官”升级的资料整理、架构说明和验证结论。

## 维护约定

- 原始论文 PDF、网页快照、截图和大体积抓取数据放在 `raw/` 或 `downloads/`，不提交到 Git。
- 可以提交整理后的 source cards、调研摘要、引用索引和产品设计说明。
- 每条资料卡必须保留可追溯 URL，不能只写“网上资料显示”。
- 资料卡的目标是辅助面试追问，不是做百科知识库；优先保留能生成高质量追问的问题模板、反模式和评分证据。
- 与产品范围或工程实现有关的决策同步更新 `target/PLAN.md`。
