# Frontend AGENTS.md

## 职责

`frontend/` 负责 AI 模拟面试官三入口训练工作台和 demo 展示体验。

## 开发约定

- UI 保持干净、现代、适合公网 demo。
- 默认后端地址为 `http://localhost:8000`，可通过 `VITE_API_BASE_URL` 覆盖。
- 保持组件结构简单，不引入复杂状态管理。
- 页面文案默认使用中文。
- 首版不加入登录、数据库或复杂路由；核心路径是选择训练入口、连续追问、查看追问依据、结束复盘。
- 首页入口固定为八股知识点、简历经历、手撕代码；三类训练的上下文和会话要隔离。
- `/training/message` 会返回资料依据字段和可选训练项；前端要用轻量 UI 展示，不把它做成复杂知识库浏览器。
- 交互改动需要跑 `npm run build` 和 `npm run test:e2e`。
